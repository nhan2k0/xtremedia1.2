## Installation
*Tested on XAMPP 1.6.5 (PHP 5.2.5, MySQL 5.0.51, phpMyAdmin 2.11.3)*
- Create new database
- Import `data.sql`
- Configure database in `includes/config.php` for correct database username, db name
