Các thay đổi trong bản 1.2 này :
Sửa :
+ Lỗi login, thay skin ( do không setcookie được )
+ Lỗi refresh
+ Lỗi search ( hiển thị một đoạn SQL )
+ Lỗi vẫn play được Singer + Album dù chưa đăng nhập
+ Lỗi hiển thị sai nội dung Lyric
+ Lỗi playlist
+ Lỗi xuất hiện thẻ <tr> đầu file khi View Source
+ Lỗi này hình như không có ai phát hiện ra thì phải ( vì không thấy báo ) : khi login xong trang Web sẽ trở về trang trước đó. Nếu bạn nhấn vào bất kỳ Link nào thì nó sẽ load lại một lân nữa.
+ Phần comment đã được hiển thị ra ngoài ( theo đa số yêu cầu của mọi người ), hỗ trợ tất cả Emoticons của Yahoo! Messenger nhưng không hiện ra để chọn ( vì lười quá nên không làm, các bạn test lại xem có OK không ), chống Flood Comment ( User chỉ được post tối đa 1 comment trong vòng 2 phút cho 1 bài hát, còn Admin + Mod thì spam nhiêu cũng được )

HƯỚNG DẪN CÀI ĐẶT :
1. Import file data.sql vào Database
2. Vào file includes/config.php sửa thông số cho phù hợp
3. Đăng nhập vào Admin CP ( /admin ) với username + pwd đều là admin. Đến phần Cấu hình --> Link Web, sửa lại cho đúng ( nếu không sẽ bị lỗi Login do không set cookie được )
4. Xong roài

HƯỚNG DẪN UPDATE TỪ 1.1 LÊN 1.2 : ( không cần đọc nếu bạn cài đặt mới )
1. Upload source mới.
2. Run update/111.php. File update này cũng sẽ sửa lỗi không thể phân quyền MOD được.
3. Đăng nhập vào Admin CP ( /admin ). Đến phần Cấu hình --> Link Web, sửa lại cho đúng ( nếu không sẽ bị lỗi Login do không set cookie được )

MỘT SỐ CHÚ Ý NỮA :
1. Nếu vẫn bị lỗi không login được, hoặc không tìm thấy file ... thì bạn hãy vào file config.php, tìm $setCookieType, sửa lại thành 2.
2. Nếu bạn vẫn xài file play_info.html của skin cũ thì sẽ không thể hiện comment ra được.
